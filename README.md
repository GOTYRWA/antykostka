# Antykostka Mod

## Installation
To install the Antykostka mod, follow these steps:
1. Download the latest version of the mod from the [releases page](https://github.com/GOTYRWA/antykostka/releases).
2. Extract the contents of the downloaded zip file into your game's `mods` folder.
3. Ensure you have any necessary dependencies installed as outlined in the mod documentation.

## Usage
Once the mod is installed:
1. Launch the game.
2. Navigate to the mods menu and enable Antykostka.
3. Start a new game or load an existing save to enjoy the features of the Antykostka mod.
